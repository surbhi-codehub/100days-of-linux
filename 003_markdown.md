# Title: Creating Directories with mkdir

## 1. Create a directory named linux_practice in your Home directory.
### Ans cd ~ && mkdir linux_practice
---
## 2. Move into the linux _practice directory and display your current location.
### Ans cd linux_practice && pwd
---
## 3. Inside linux_pratice, create a directory and display named day03.
### Ans mkdir day03
---
## 4. Return to your Home directory and list its contents.
### Ans cd ~ && ls
---
## 5. Create two directories named data and scripts inside linux_practice using a single command.
### Ans cd ~/linux_practice && mkdir data scripts
---
